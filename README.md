# 哔哩哔哩自动发送弹幕进行每日打卡（A-SOUL成员）

cookie获取方法可参考(https://github.com/XiaoMiku01/bili-live-heart/blob/master/doc/bili.md)

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### Start project

```bash
npm start
```
